🧪 Test Thống Kê Đơn Hàng

📋 Hướng dẫn test

1. Đảm bảo Laravel server đang chạy: php artisan serve

2. Truy cập trang admin: http://localhost:8000/admin

3. Kiểm tra trang thống kê: http://localhost:8000/admin/stats

🔧 Test Command

Chạy command để cập nhật thống kê:

php artisan orders:update-stats

📊 Kiểm tra Views

  • ✅ view_monthly_revenue - Doanh thu theo tháng
  • ✅ view_paid_revenue - Doanh thu đã thanh toán
  • ✅ view_order_status_count - Số lượng đơn hàng theo trạng thái
  • ✅ view_top_products - Sản phẩm bán chạy
  • ✅ view_top_coupons - Mã giảm giá được sử dụng nhiều

🎯 Test Cases

  1. Tạo đơn hàng mới - Kiểm tra xem có xuất hiện trong thống kê không
  2. Chuyển trạng thái đơn hàng - Kiểm tra event listener có hoạt động không
  3. Kiểm tra điểm tích lũy - Xem điểm có được tích khi đơn hàng hoàn thành
  4. Kiểm tra trừ kho - Xem kho có bị trừ đúng không

✅ Đã sửa

  • ❌ Lỗi variant_name → ✅ Sử dụng product->name
  • ❌ Lỗi total_usage → ✅ Sử dụng used_count
  • ❌ Trừ kho trùng lặp → ✅ Chỉ trừ khi thanh toán thành công
  • ❌ Thống kê không cập nhật → ✅ Tự động cập nhật

🔍 Debug Info

Nếu vẫn có lỗi, kiểm tra:

  • Log file: storage/logs/laravel.log
  • Database connection
  • Views có tồn tại không
  • Permissions của user
